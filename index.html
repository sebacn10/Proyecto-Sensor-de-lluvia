PRESENTACION

Este proyecto consiste en un techo corredizo automatizado
diseñado para proteger la ropa colgada de la lluvia. Utilizando
un sensor de agua, el sistema al detectar la presencia de
humedad activa el mecanismo para desplegar el techo, asegurando que la ropa se mantenga seca.
