Información del Sitio

Preguntas Frecuentes

¿Cómo funciona el techo corredizo?

El sistema utiliza un sensor de agua que detecta la lluvia. Cuando se activa, envía una señal al
Arduino que pone en movimiento un sistema de poleas para cerrar el techo automáticamente.

¿Qué pasa cuando deja de llover?

El sistema puede configurarse para abrir el techo nuevamente después de un tiempo o de forma manual.

¿Es necesario mantenimiento?

Sí, se recomienda revisar las poleas, cables y sensores periódicamente para asegurar su correcto funcionamiento.

¿Se puede usar en exteriores?

Sí, pero los componentes electrónicos deben estar protegidos contra la humedad.
